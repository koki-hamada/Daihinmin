◎概要
第3回 UEC コンピュータ大貧民大会(UECda-2009)に向けて、オリジナルの
クライアント・ソフトウェアをJAVAで開発する方向けの開発キットです。

◎内容
daihinmin.jar:
  全てのプログラムを含むJarファイルです．
  クライアントを作成するときはこのファイル内のパッケージをimportします．
  対戦を行うときは，このプログラムを実行します．

man-*:(*にはマニュアルのバージョン番号が入ります)
  UECda-2008のルールや、サーバーの使い方、クライアントの作成方法など
  必要な情報が記載されたマニュアルです。
  manual.html もしくは index.html が目次ページとなっています。

doc/index.html:
  javadocです．daihinmin.jar内のクラスの詳細が記述されています．

README.txt :
  このファイルです。

◎実行環境
本プログラムはJVM1．5以上で動作します．バージョンの確認はコマンドプロンプトで
以下のコマンドを実行してください．
java -version

*注意*
大会ではJVM1．5を使用します．
クライアント作成者は，JVM1．5上で動くようにコンパイルしてください．

◎使用方法
*対戦するとき*
java -jar daihinmin.jar [-mode mode][-port port_number][-players player ...]
  -mode mode モードを設定します．aloneとするとスタンドアロンモードで実行します．
  
  -players player ... 対戦に使用するプレイヤークラスを指定します．必ず5つ指定する必要があります．
BotPlainとするとJAVA版標準クライアントが対戦に参加します．
  
  -port port_number 通信に使用するポート番号を指定します．

*開発するとき*
BotHoge.javaをクライアントとして開発していて，それをコンパイルする場合，
javac -cp daihinmin.jar BotHoge.java
としてください．

詳しくは，付属のマニュアル dhm_doc*.pdf 第9章クライアント作成の準備(Java版) をご参照下さい．
